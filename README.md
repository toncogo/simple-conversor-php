💸 Conversor de Real para Dólar (PHP + API BACEN)

Este é um projeto simples desenvolvido para praticar a integração entre tecnologias web básicas e o consumo de APIs externas. O programa realiza a conversão de valores de Real (BRL) para Dólar (USD) utilizando a cotação em tempo real fornecida pelo Banco Central do Brasil.
🚀 Funcionalidades

    Consumo de API: Busca automaticamente a cotação atualizada do dia através da API do Banco Central.

    Tratamento de Erros: Verificação de disponibilidade da API e validação básica de dados.

    Interface Limpa: Desenvolvida com HTML5 e estilizada com CSS3.

🛠️ Tecnologias Utilizadas

    HTML5: Estruturação da interface.

    CSS3: Estilização visual.

    PHP: Lógica de back-end, tratamento de JSON e integração com a API.

⚙️ Como executar

    Certifique-se de ter um servidor local instalado (como XAMPP, WAMP ou Laragon).

    Clone este repositório na pasta htdocs (ou equivalente do seu servidor).

    Inicie o serviço do Apache.

    Acesse localhost/nome-da-pasta no seu navegador.

    Nota: Este projeto foi criado para fins de estudo sobre manipulação de dados externos e lógica de programação em PHP.
